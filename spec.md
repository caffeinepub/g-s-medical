# Specification

## Summary
**Goal:** Revert the entire G&S Medical application to exactly the state it was in at Version 9, removing all changes introduced in Version 10 and beyond.

**Planned changes:**
- Restore frontend code (components, pages, styles, routes) to match Version 9 exactly
- Restore backend actor (data models, functions, logic) to match Version 9 exactly
- Remove all features, animations, styles, and backend changes introduced after Version 9

**User-visible outcome:** The application reflects Version 9 exactly — all pages, routes, and functionality from Version 9 work correctly, and nothing added in later versions is present.
